Bienvenido/a a mi proyecto basado en Django. En esta página encontrarás una amplia gama de funcionalidades para demostrar los conocimientos que tengo al usar este FrameWrok con Python.
Incialmente, las habilidades que deseo demostrar son las operaciones CRUD. Puedes crear un usuario (en la opcion Login del menú) donde una vez creado, puedes tener acceso a diferentes rutas, las cuales tambien estan protegidas y solo se tendrá acceso si estas logado con usuario.

Si deseas, puedes crear un usuario para que puedas ver de manera personalizada cada pestaña, de lo contrario, puedes usar el usuario de prueba que dejo en a continuacion.

Un vez logueado, podrás:
-Crear una tarea a travez de un formulario y marcarla como importante (opcional)
-Una vez creada, puedes visualizarla en la lista de: Pending Task
-Puedes ingresar a esa tarea y tendras las opciones de: Editar la tarea y guardar, Completar la tarea (se elminará de tareas pendientes y cambiará a la pestaña de tareas completadas), o si ya no deseas continuar con esta tarea, puedes eliminarnar.
-En la pestaña: Completed Task, podras ver todas las tareas que te propusiste y que ya has completado.
-Cuando hayas finalizado con interacion en la pagina, podras salir con la pestaña Logout.
